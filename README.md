##  For Frontend Run Locally

### Install Dependencies
```bash
npm install
```

### Start Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```


## For Backend Run Locally

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Start Development Server
```bash
uvicorn app.main:app --reload
```

### Run Test Suite
```bash
pytest
```
